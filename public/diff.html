Differences between Coursefinder and Murphy Online

Subject Selection

Murphy Online uses a cumbersome text select box in order to filter classes by subject matter. Selecting multiple classes requires the user to unintuitively control click on multiple subjects within the select box. CourseFinder uses an expandable list of check boxes, making it simple for the user to understand and change the selected subjects.

Wishlist

Our registration system allows users to save a wishlist of classes they would like to register for in the future, providing easy access to these classes at the time of registration. They can even register classes from this view. Users save time and stress during the registration process, as they can quickly access the classes they have planned to take. Murphy Online does not provide this functionality.

Schedule Viewer

Our registration system allows the user to quickly navigate to classes that they are currently registered for, displaying valuable information like the class's time, name, and professor. Murphy Online's schedule viewer is not as comprehensive for planning one's schedule.

Course Highlighting

In our registration system, classes that the student is registered for will appear in green, and classes that the student has waitlisted will appear in yellow.

Websocket Updates

Our registration system will update the number of registered students for a given class in real time without page refreshing by utilizing web sockets. It will even update the GUI immediately to prevent users from registering. Murphy Online only updates the number registered users when the page is refreshed, leading to the potential situation of registering for a course only to discover that it was filled by other students on submission.

Waitlist

Murphy Online's waitlist system requires the student to search for the class,write down the CRN, and attempt to register for it with the CRN on the Add/Drop page. During this whole process, the student will not know if the class is available or has a waitlist, as the class will appear to be closed. On the other hand, our registration system has a unified waitlist system. When a course is full, our search system will display an option to "Add to Waitlist." Students who are on the waitlist will be automatically moved to the list of registered students when a spot becomes available. In addition, classes that the user has on their waitlist show up in the search with a yellow background, while registered classes appear in green.
